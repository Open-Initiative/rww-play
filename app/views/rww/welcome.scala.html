About

This application intends to implement a suite of tools to build a read-write-web

  • The Linked Data Platform (LDP) defined at the W3C
  • WebID Certificate Creation
  • WebID Authentication over TLS
  • A WebID Identity Provider
  • Web Access Control
  • Account creation and friend management
  • Blog engine
The aim is to have these pieces working together to build a use case, and later seperate the pieces into libraries.
